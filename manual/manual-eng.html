Installation Manual:

The Cardabase application is available in the F-Droid store or via GitHub releases only for devices with Android 5.0 and higher. Installation for other platforms is not supported. Google Play Store installation will fail due to the installation of a third-party application (in this case, Cardabase).

F-Droid:
  1. Use the F-Droid store for installation.
  2. After installation, the application is ready.
  3. The F-Droid store will also be used for automatic updates.
GitHub:
  1. Download the application as an .apk file via GitHub releases:
    • Universal - if you don't know which one to choose (contains all ABIs, but is larger, approx. 57 MB).
    • ARM 64-bit v8a - for modern devices (recommended, approx. 20 MB).
    • ARM 32-bit v7a - for older and cheap devices (hardly used today, approx. 18 MB).
    • x86 64-bit - for devices with x86 processors (e.g., some tablets and computers, approx. 21 MB).
  2. Then find it in the Downloads folder.
  3. Click on it and select install.
  4. After installation, the application is ready.
  5. For application updates, you will need to use the GitHub releases again.
