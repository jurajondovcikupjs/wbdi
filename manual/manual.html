Návod na inštaláciu:

Aplikácia Cardabase je dostupná v obchode F-Droid alebo cez GitHub vydania len pre zariadenia s Androidom 5.0 a vyšším. Inštalácia pre iné platformy nie je podporovaná. Google Play sken zlyhá kvôli inštalácii aplikácie tretej strany (v tomto prípade Cardabase).

F-Droid:
  1. Použite obchod F-Droid pre inštaláciu aplikácie.
  2. Po inštalácii je aplikácia pripravená.
  3. Obchod F-Droid bude použitý taktiež pre automatické aktualizácie.
GitHub:
  1. Stiahnite si aplikáciu ako .apk súbor cez GitHub vydania:
    • Univerzálna - ak neviete, ktorú zvoliť (obsahuje všetky ABI, ale je väčšia, cca 57 MB).
    • ARM 64-bit v8a - pre moderné zariadenia (odporúčané, cca 20 MB).
    • ARM 32-bit v7a - pre staršie a lacné zariadenia (dnes už takmer nepoužívané, cca 18 MB).
    • x86 64-bit - pre zariadenia s procesorom x86 (napr. niektoré tablety a počítače, cca 21 MB).
  2. Následne ju nájdite v priečinku Stiahnuté, resp. Downloads.
  3. Kliknite na ňu a vyberte inštalovať.
  4. Po inštalácii je aplikácia pripravená.
  5. Pre aktualizácie aplácie musíte znova použiť GitHub vydania.
