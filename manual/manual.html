Návod na inštaláciu:

F-Droid:
  1. Použite obchod F-Droid pre inštaláciu.
  2. Po inštalácii je aplikácia pripravená.
GitHub:
  1. Stiahnite si aplikáciu ako .apk súbor cez GitHub vydania.

    2. Verzie ABI:

    • Univerzálna - ak neviete, ktorú zvoliť (obsahuje všetky ABI, ale je väčšia).
    • ARM 64-bit v8a - pre moderné zariadenia (odporúčané).
    • ARM 32-bit v7a - pre staršie a lacné zariadenia (dnes už takmer nepoužívané).
    • x86 64-bit - pre zariadenia s procesorom x86 (napr. niektoré tablety a počítače).
  2. Následne ju nájdite v priečinku Stiahnuté, resp. Downloads.
  3. Kliknite na ňu a vyberte inštalovať.
  4. Po inštalácii je aplikácia pripravená.
