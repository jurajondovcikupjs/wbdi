Cardabase

It is a FOSS project that tries to provide a simple and secure application for managing club cards. Simply scan the card, fill in the details and keep them in a secure storage. No fees, forever.



Download via F-Droid Download via GitHub
SOURCE CODE

Features:

  • Pleasant user interface,
  • Secure storage using AES-256 encryption,
  • Ability to secure cards with a password,
  • Support for a large number of barcodes,
  • Completely offline, no internet required,
  • Lightweight for storage,
  • Open source code,
  • Completely free, without any fees, forever.
