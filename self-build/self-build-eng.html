Self-build:

Guide:

  1. Download the source codes from GitHub:
    git clone https://github.com/GeorgeYT9769/cardabase-app.git
  2. Change the working directory to the downloaded repository:
    cd cardabase-app
  3. Install necessary tools and dependencies:
    flutter pub get
  4. Run in the command line:
    flutter build apk (--split-per-abi optional)
  5. After completing the process, you will find the generated APK file in the directory:
    build/app/outputs/flutter-apk/app-release.apk
    and install it on your device using ADB:
    adb install build/app/outputs/flutter-apk/app-release.apk.

    App signing is enabled. In the /cardabase-app/android folder, you need a signing key and a key.properties file, otherwise compilation will fail. If you want to sign the app using a debug key, you can change THIS LINE to debug.


Used tools and their versions:

  • Flutter 3.38.5 (always bundled)
  • Java JDK-21
  • Gradle: 8.14.2

    Java and Gradle must be compatible with each other. See this Compatibility Matrix.

You might need:

  • Signing key and key.properties file
  • Android SDK
  • Specify JAVA_HOME in gradle.properties, add the line:
    org.gradle.java.home="C:\\path\\to\\the\\jdk"
  • Change the Gradle version in gradle-wrapper.properties:
    distributionUrl=https\://services.gradle.org/distributions/gradle-X.X.X-bin.zip
