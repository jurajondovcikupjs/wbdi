Vlastné zostavenie:

    Návod na vlastné zostavenie aplikácie Cardabase:

  1. Stiahnite si zdrojové kódy z GitHubu:
    2. git clone https://github.com/GeorgeYT9769/cardabase-app.git
  2. Zmeňte pracovný adresár na stiahnutý repozitár:
    3. cd cardabase-app
  3. Inštalujte potrebné nástroje a závislosti:
    4. flutter pub get
  4. V príkazovom riadku spustite:
    5. flutter build apk (--split-per-abi dobrovoľné)
  5. Po dokončení procesu nájdete vygenerovaný súbor APK v adresári build/app/outputs/flutter-apk/app-release.apk.
    6. adb install build/app/outputs/flutter-apk/app-release.apk

    Podpisovanie aplikácie je zapnuté. V priečinku /cardabase-app/android potrebujete podpisový kľúč a súbor key.properties, inak kompilácia zlyhá. Ak chcete aplikáciu podpísať pomocou ladiaceho kľúča, môžete TENTO RIADOK zmeniť na debug.


    Použité nástroje a ich verzie:

  • Flutter 3.38.5 (vždy pribalená)
  • Java JDK-21
  • Gradle: 8.14.2

    • Java a Gradle musia byť navzájom kompatibilné. Pozri tento Compatibility Matrix.

    Možno budete potrebovať:

  • Podpisový kľúč a súbor key.properties
  • Android SDK
  • Špecifikovať JAVA_HOME v gradle.properties, pridajte riadok:
    • org.gradle.java.home="C:\\path\\to\\the\\jdk"
  • Zmeniť verziu Gradle v gradle-wrapper.properties:
    • distributionUrl=https\://services.gradle.org/distributions/gradle-X.X.X-bin.zip
